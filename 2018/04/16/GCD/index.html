BZOJ GCD

发表于 2018-04-16 | 分类于 数论

Description

给定整数N，求$1<= x,y<= N$且$gcd(x,y)$为素数的数对$(x,y)$有多少对.
$N<=10^{7}$

Solution

可以从小到大枚举素数$p$,若$gcd(x,y)=p$,则令$x=p\cdot a$同时$y=p\cdot b$,因为$gcd(x,y)=1$,所以有$gcd(a,b)=1$,否则$gcd(x,y)\neq p$,得出$a$和$b$互质,所以对于每一个整数$a=\frac{N}{p}$,所有小于它的数的$k$的$\phi(k)$都成立,又因为&x,y&可以互换位置,所以答案就是$\sum_{i=1}^{\frac{N}{p}}\phi(i) \cdot 2 - 1$,这里的减$1$是因为$a=1,b=1$被多算了一次。
还要注意数组的大小,素数表开一半就行了,用int,其余用long long
预处理欧拉函数要用线性筛法,不然会TLE
